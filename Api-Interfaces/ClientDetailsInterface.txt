3) Полный поиск (GET)// URL: /solid?q=  (где параметр q - это строка. Поиск должен осуществляться по принципу "Полное совпадение q с полем detailNumber")
SolidSearchResponse // Респонс за запрос полного поиска
{
	SolidSearchResult[] result; // пустой массив, если нет результатов
}
SolidSearchResult
{
	int (number) internalId; // наш внутренний Id
	enum (string) type; // origin, analog
	string detailNumber;
	string originalDetailNumber;
	string detailName;
	Decimal (number) price;
	string delivery;
	string description;
	int (number) quantity;
}

4) Получение списка вариантов доставки (Get) URL: /getDeliveryMethods
GetDeliveryMethodsResponse
{
	DeliveryMethod[] methods;
}

DeliveryMethod
{
	int deliveryId; // наш внутренний ID доставки
	enum(String) locationType; // moscow, russia
	enum(String) deliveryType; lightweight, largeweight
	string deliveryName; // Название доставки
	Decimal deliveryPrice; // Стоимость, которая добавляется к заказу. Если 0 - доставка бесплатная. Если поле отсутствует - обсуждается индивидуально
}
